# FundFlow


![screenshot](./Screenshot%20(12).png)

A modern crowdfunding platform built with Next.js, enabling users to create, manage, and support fundraising campaigns.

## Features

- **Campaign Creation**: Create and manage crowdfunding campaigns
- **Secure Donations**: Support campaigns through secure payment processing
- **Real-time Updates**: Track campaign progress and funding goals
- **Responsive Design**: Fully responsive interface for all devices
- **User Authentication**: Secure user management and authentication

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org) - React framework for production
- **Styling**: Modern CSS with optimized fonts ([Geist](https://vercel.com/font))
- **Font Optimization**: Automatic optimization using `next/font`
